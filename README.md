# 🛒 Ecommerce Purchase Prediction — Python ML Project

ML project predicting customer purchase behavior 
using classification and clustering models.

## What I Built
- Probability analysis of purchase conversion
- Hypothesis testing — T-Test, Z-Test, Chi-Square
- CLT demonstration
- Logistic Regression — purchase prediction
- KNN Classifier — K optimization (1-10)
- KMeans Clustering — 4 customer segments
- Silhouette score evaluation

## Key Findings
- Only 22% of visitors actually purchase
- Time on site alone does not predict purchase
- Session duration and purchase are statistically linked
- 4 distinct customer segments identified

## Tools
Python, Pandas, Scikit-learn, Matplotlib, Seaborn
